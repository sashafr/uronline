Specify a property

Select a property to search by from the "Property" menu. This will specify that you would like to search only that property for your Search Terms.

Specify a search type

  • "contains" - selecting a property and "contains" will check for any object which contains your Search Terms for the specified property. This is the default search.
  • "like" - selecting a property and "like" allows you to use wildcards in your search. To represent a single character, use "?", ie. "te?t". To represent 0 or more characters, use "*", ie. "te*t".
  • "equals" - selecting a property and "equals" will check for any object which contains EXACTLY your Search Terms for the specified property.
  • "blank" - selecting a property and "blank" will check for any object which does not have any data for the specified property.
  • "starts with" - selecting a property and "starts with" will check for any object which contains words that begin with your Search Terms.
  • "ends with" - selecting a property and "ends with" will check for any object which contains words that end with your Search Terms.
  • "greater than", "less than", etc - selecting one of these search types will treat your Search Terms as a number and search for objects which contain the specified property and that its value evaluates to true using your Search Terms
